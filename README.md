# Info in slot
Mod was inspired by Map in Slot https://github.com/Crec0/map-in-slot

This mod extracts information from the Tooltip sent by the DonutSMP server without the need to hover mouse over it, then uses that information to display a list of stats onto the inventory hotbar Slot

List of Stats that are displayed
- Self Destruct or Expire Timer
- Auction or Order Item Price
- Order Delivered Count (for personal orders)
- Order Count available to be Delivered (for other player orders)
information is obtain by taking the order request count and subtracting it by the already delivered count for example  25/100 Delievered would Display as 75

Configuration Options:
Ability to change
- Color
- Bold Font
- Text Size (Small, Medium, (Large for Timer only))
- Toggle Background (dark gray square behind text)
- Position (Top, Middle, Bottom)
for all 4 Categories 
Timer Display
Auciton Price
Order Price
Order Display

also includes Color Seperator option for Expiration Timer
2+Days, Day, Hours, Minutes
Magenta, Aqua, Gold, Red

all can be changed either thru (Mod Menu) or thru Commands for those who arent using Mod Menu
or just want to quickly type it out
/slotinfo

Control Keybinds
- Toggle Mod on & Off (unbound)
- Cycle Timer Size (unbound) 
when keybind pressed cycles Expire Timer character size beween Small, Medium, Large
- Toggle Hours only (h)
converts times like 2d 20h to 68h
